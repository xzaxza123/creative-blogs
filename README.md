# 网站介绍

框架:vue3 -- 虽然说是vue3，但是没有使用vue3的语法糖就是setup,不太熟练

主要插件：turn.js(一个基于jquery用来实现3d翻书效果的插件)

UI框架：Element Plus(使用了少量的组件，例如：模态框、按钮、图片上传等)

富文本编辑器：wangeditor5

服务端API：PHP(对于PHP知识知道的不太多，这个网站只是使用了对于数据库的操作，以及图片的上传)

PHP服务器开发环境：xmapp(本地)

数据库：MySQL
